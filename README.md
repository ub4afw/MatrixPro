# 🧮 MatrixPro — Генератор таблиц умножения

**MatrixPro** — это простая и удобная программа на Python, которая позволяет гибко строить таблицы умножения (матрицы Пифагора) для любых чисел. Проект идеально подходит для учебных целей, быстрой проверки навыков счета или визуализации таблицы умножения в удобном формате.

## ✨ Возможности

- Построение таблицы умножения для заданного диапазона чисел (например, от 1 до 10).
- Настройка размера таблицы через конфигурационный файл `multiplication_config.json`.
- Чистый и читаемый вывод в консоль в виде форматированной матрицы.
- Легкость модификации кода под свои нужды.
